# Diedericks Dobermanns — Breeding Reference Guide
*Quick reference for breeders using the app*

---

## Heat Cycle Prediction

### How the system predicts the next heat

| Situation | What the system uses |
|-----------|---------------------|
| First heat recorded | Dobermann breed average: **180 days** |
| 2+ heats recorded | That female's **personal average** cycle length |

The prediction updates automatically every time you confirm a real heat. The more heats you record, the more accurate the prediction becomes for that specific female.

### Dobermann Heat Phase Guide

| Phase | Duration | What happens |
|-------|----------|--------------|
| Proestrus | ~9 days | Swelling, bloody discharge. Female attracts males but won't accept mating. |
| Estrus | ~7 days | Discharge lightens. Female accepts mating. **Fertile window.** |
| Diestrus | ~75 days | Body prepares for pregnancy or false pregnancy. |
| Anestrus | ~89 days | Rest period. No hormonal activity. |

### Progesterone Testing Guide

| Value (ng/mL) | Meaning | Action |
|---------------|---------|--------|
| < 2 | Baseline — no ovulation yet | Too early to breed |
| 2 – 5 | Approaching ovulation (LH surge) | Test every 1–2 days |
| 5 – 15 | Ovulation occurred | Breed now |
| > 15 | Peak — optimal breeding window | Best time to breed |

**Best practice:** Start testing from Day 5 of heat. Test every 2 days until value rises above 5, then breed.

### Optimal Breeding Window
- Natural: ovulation day + 2 to + 4 days
- AI (fresh/chilled): ovulation day to + 2 days
- AI (frozen): ovulation day + 1 to + 3 days

## Whelping Date Calculation

### From Mating Date (less accurate)

| Milestone | Calculation |
|-----------|-------------|
| Earliest whelp | Mating date + 57 days |
| Expected whelp | Mating date + 60 days |
| Latest whelp | Mating date + 65 days |

### From Ovulation Date (more accurate — use when progesterone tested)

| Milestone | Calculation |
|-----------|-------------|
| Earliest whelp | Ovulation date + 60 days |
| Expected whelp | Ovulation date + 63 days |
| Latest whelp | Ovulation date + 66 days |

**Always use ovulation date when available — it is significantly more accurate than mating date alone.**

---

## Puppy Go-Home Date

Calculated automatically from the **actual birth date** (not estimated).

| Milestone | Calculation |
|-----------|-------------|
| Earliest go-home | Birth date + 8 weeks (56 days) |
| Standard go-home | Birth date + 9 weeks (63 days) |
| Latest go-home | Birth date + 10 weeks (70 days) |

**Note:** Before actual birth, the app shows estimated go-home dates based on expected whelp. Once you record the real birth date, go-home dates recalculate automatically.

---

## What Updates Automatically

| You enter | System auto-calculates |
|-----------|----------------------|
| Heat start date | Proestrus end, estrus start, estimated ovulation, predicted next heat |
| Mating date | Whelp date range, estimated go-home range |
| Ovulation date (from progesterone) | More accurate whelp date range, go-home range |
| Actual birth date | Exact go-home date range (replaces estimates) |
| 2nd confirmed heat | Switches from breed average to personal cycle average for predictions |

## Female Cycle Status in App

| Indicator | Meaning |
|-----------|---------|
| 🔴 Red dot — "In Heat Day X" | Currently in active heat |
| 🟡 Gold — "Next heat: [date]" | Predicted upcoming heat |
| 🟠 Orange — "Overdue" | Predicted heat date has passed, no heat confirmed yet |
| ⚪ Grey — "No history" | No heats recorded yet for this female |
